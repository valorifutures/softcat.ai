Visual comparison of context window sizes for the major AI models. Slide the usage control to see how each model handles different fill levels.

Shows warning indicators at 80% capacity (yellow) and full capacity (red). Bars scale relative to the largest window so you can see the real differences between a 128K model and a 1M model.

Useful for planning which model to use when you need to fit large documents, long conversations, or multi-file codebases into a single call.
