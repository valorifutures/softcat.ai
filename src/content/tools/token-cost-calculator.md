Paste your prompt text or a full conversation, pick a model, and see the estimated cost before you send.

Two modes: single prompt (input + output) and conversation (multi-turn with User/Assistant markers). The conversation mode shows per-turn token counts and costs in a table, so you can see where the tokens are going.

Uses real API pricing from public rate cards. Token estimates are approximate, within about 15% of actual tokeniser counts.
